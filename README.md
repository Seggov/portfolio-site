# Pedro Segovia | Portafolio

Portafolio profesional estático enfocado en backend, datos y automatización.

## Sitio

- Portafolio: <https://seggov.github.io/portfolio-site/>
- Perfil: <https://github.com/Seggov>

## Características

- Diseño responsive y navegación móvil.
- Sección de proyectos destacados con ventanas de detalle accesibles.
- Laboratorio conectado con repositorios públicos.
- Sección de intereses y stack tecnológico.
- Navegación por teclado, modo de movimiento reducido y mejora progresiva.
- Metadatos para buscadores y redes sociales.
- Imagen de perfil optimizada para web.

## Estructura

```text
.
├── index.html
├── styles.css
├── script.js
├── robots.txt
├── sitemap.xml
└── imagenes/
```

El sitio no necesita proceso de compilación: GitHub Pages publica directamente los archivos de la rama `main`.
